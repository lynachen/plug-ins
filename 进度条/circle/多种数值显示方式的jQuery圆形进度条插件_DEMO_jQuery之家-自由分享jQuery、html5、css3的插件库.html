Each example is accompanied by JavaScript and CSS code. You can switch between plain JS and jQuery snippets.

	max: 100,
	value: 60,
	textFormat: 'percent',
});
$('.progress').circleProgress({
	max: 100,
	value: 60,
	textFormat: 'percent',
});
